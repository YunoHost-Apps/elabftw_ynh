eLabFTW est un gestionnaire de cahiers de laboratoire électroniques destiné aux équipes de recherche. Il vous permet de stocker et d'organiser facilement vos expériences de recherche. Il dispose également d'une base de données dans laquelle tout type d'objets (anticorps, plasmides, lignées cellulaires, boîtes, etc.) peut être stocké. Il est accessible via un navigateur. Plusieurs équipes de recherche peuvent être hébergées sur la même installation. Cela signifie qu'eLabFTW peut être installé au niveau de l'institut et héberger tous les membres de l'équipe au même endroit. C'est ce qui se fait à l'Institut Curie et dans plusieurs autres centres de recherche à travers le monde.
